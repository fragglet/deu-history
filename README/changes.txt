Changes from DETH 2.4 --> DETH 2.5 (Compiled 18th July '95)
-----------------------------------------------------------

Bugs Fixed
----------
Search (F4) would crash when item not found.
(E)dit level could only handle 32 levels not 38 as in Ultimate Doom

Additions
---------
Change Sectors Floor/Ceiling or Light level by any amount - very useful

Hot Keys:-
Objects must be clicked (selected) for the hot keys to work, not just
highlighted with the mouse.

Hot Keys All Modes
~~~~~~~~~~~~~~~~~~
CTRL+T - Next Free (T)ag

Hot Keys Line Mode
~~~~~~~~~~~~~~~~~~
CTRL+2 - Toggle (2) sided flag
CTRL+A - Split LineDef and (A)dd a Sector
CTRL+B - Toggle (B)locks sound flag
CTRL+D - (D)elete LineDefs and Join Sectors (same as CTRL+J)
CTRL+F - (F)lip LineDef
CTRL+H - Toggle (H)idden on map flag
CTRL+J - Delete LineDef and (J)oin Sectors (same as CTRL+D)
CTRL+L - Toggle (L)ower texture unpegged flag
CTRL+M - Toggle i(M)passible flag
CTRL+N - Set LineDef type to (N)ormal
CTRL+O - Toggle already (O)n map flag
CTRL+P - Toggle monsters can't (P)ass flag
CTRL+R - Toggle sec(R)et flag
CTRL+S - (S)wap SideDef
CTRL+U - Toggle (U)pper texture unpegged flag
CTRL+V - Split LineDef and Add a (V)ertex
CTRL+X - Align Textures on (X) without Texture Checking
     X - Align Textures on (X) with Texture Checking
CTRL+Y - Align Textures on (Y) without Texture Checking
     Y - Align Textures on (Y) witht Texture Checking

Hot Keys Vertex Mode
~~~~~~~~~~~~~~~~~~~~
CTRL+A - Add LineDef and (A)dd a Sector
CTRL+D - (D)elete Vertex and Join LineDefs (same as CTRL+J)
CTRL+J - Delete Vertex and (J)oin LineDefs (same as CTRL+D)
CTRL+M - (M)erge Several Vertices and Join LineDefs

Hot Keys Things Mode
~~~~~~~~~~~~~~~~~~~~
CTRL+D - Toggle (D)eaf flag
CTRL+M - Toggle (M)ultiplayer flag

Hot Keys Sector Mode
~~~~~~~~~~~~~~~~~~~~
CTRL+N - Set Sector type to (N)ormal




Changes from DETH 2.3 --> DETH 2.4 (Compiled 6th July '95)
----------------------------------------------------------

Bugs Fixed
----------
None

Additions
---------
(F4) Search facility now improved, Search or Search & Select any Thing type.
(F)ilter, used to show/hide things by difficulty class and Deathmatch flag.
Drawing of grid is now slightly faster.
Menus now stay when clicked with mouse.
How close you have to be to a LineDef to split it when Inserting Vertices
	is now definable. see the deth.ini file and F5 - Preferences.
Doom.cfg updated to handle Ultimate Doom.
Heretic.cfg updated to handle hidden E4M1.
LineDefs and Sectors colors are now configurable in .cfg files
Auto insert of next free tag number
